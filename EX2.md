# Ex.Name: Write a CPP Program to print the sorted elements of the Array.

## Date: 13/11/2025
## Aim:
To write a C++ program to read the elements of an array and print the sorted elements.

## Algorithm:
```
1. Start the program.
2. Read the number of elements `n`.
3. Input `n` elements into the array.
4. Use the built-in `sort()` function or any sorting technique to sort the array in ascending order.
5. Display the sorted array.
6. Stop the program.
```



## Program:
```cpp
void print(int arr[]){
     printf("After Sorting the Array: \n");
    for (int i = 0; i < 5; i++)
        printf("%d ", arr[i]);
}
```
